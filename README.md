# 春秋云镜 Writeup

这个仓库只做一件事：保存我在春秋云镜靶场里的打靶记录。

## 用法

- 每打一台靶机，就在 `writeups` 目录下新建一个 Markdown 文件
- 文件里直接写过程、命令、截图说明和复盘
- 然后提交到 GitHub

## 目录

```text
.
|-- README.md
`-- writeups/
    `-- README.md
```

## 命名建议

- `靶机名.md`
- 例如：`example-target.md`

## 示例

```text
writeups/example-target.md
```

## 注意

- 上传前注意脱敏
- 不要把账号、密码、token、flag 等敏感信息直接公开
